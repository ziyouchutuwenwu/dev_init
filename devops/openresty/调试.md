# 调试

## 用法

设置 log 到 stdout

```conf
error_log /dev/stderr warn;
access_log /dev/stdout;
```

location 里面

```lua
location /demo/ {
    access_by_lua_block {
        ngx.say("uri=", ngx.var.uri)
        ngx.say("client_ip=", ngx.var.remote_addr)
        ngx.exit(200)
    }

    header_filter_by_lua_block {
        ngx.log(ngx.WARN, "header filter phase - status: ", ngx.status)
    }

    proxy_pass http://localhost:8080/;
}
```
