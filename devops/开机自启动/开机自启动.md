## 比较老的 initd 模式

vim /etc/rc.local

```sh
#!/bin/sh -e

exit 0
```

然后

```sh
reboot chmod +x /etc/rc.local

```

## systemd 模式

vim demo.service

```sh


```

启动
systemctl start demo

开机自启：
systemctl enable demo

重启：
systemctl restart demo

停止：
systemctl stop demo

状态：
systemctl status demo
