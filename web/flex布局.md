# flex 布局说明

## 最简说明

### 直属父容器

```txt
display: flex;
设置方向flex-direction
设置对齐方式：justify-content: flex-end;
```

### 子元素

```txt
设置权重flex属性
```

### 比较有用的辅助配置

```css
div {
  display: flex;
  border: #8f8f94 solid 1px;
  margin: 20px;
}
```
