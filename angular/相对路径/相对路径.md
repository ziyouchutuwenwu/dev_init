# 相对路径配置

## 相对路径配置，详情见[这里](https://github.com/angular/angular/issues/30835)，流程如下

- 修改 app.module.ts

```typescript
providers: [
    {provide: APP_BASE_HREF, useValue : '/backend' }
  ],
```

- 修改 index.html

```html
<base href="./backend" />
```

- 注意上面两个配置，一个是`相对路径`，一个是`绝对路径`，不要弄错了
- 生成发布版`ng b --prod`，然后修改 dist 目录里面的 index.html，把`<script type="module">`相关的部分，改成`/textjavascript`即可。
