# 小程序

## 安装vue-cli

```bash
npm install -g @vue/cli
```

## 创建项目

```bash
vue create -p dcloudio/uni-preset-vue vue-demo
```

- profile里面可以添加

```bash
alias uni-creator='vue create -p dcloudio/uni-preset-vue'
```

## HBuilderX导入vue创建的项目

- 发布为移动端的时候，需要HBuilderX
- mkdir xxx，然后复制vue项目src目录到xxx里面，HBuilderX导入即可。
