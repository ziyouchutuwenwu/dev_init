# govcl 教程

## 说明

- govcl 是一个跨平台的 ui 库，基于 golang， `github.com/ying32/govcl`

## 开发步骤

- `git clone github.com/ying32/govcl`, 找到 res2go，编译，放到\$PATH 所在目录

- 打开 lazarus，创建项目，ui 上拖好控件，工程属性, 配置 res2go

- 打开 goland，创建项目，把 xxx 目录里面的 go 文件复制过去，直接写业务

## 构建

- 跨平台编译

```bash
GOOS=windows GOARCH=386 go build -ldflags="-H windowsgui"
```

- 注意，运行需要 `liblcl` 这个库，可以用 github 的 release 页面下载
- `对于xp的支持，需要golang的老版本`

## 多线程回调

```golang
go func() {
    vcl.ThreadSync(func() {
        vcl.ShowMessage("这是主线程的异步对话框")
    })
}()
```
