# 其他一些设置

## html 配置子目录

```java
public void configRoute(Routes router) {
    router.setBaseViewPath("/backend");
}
```

## 自定义错误页面

- 代码如下

```java
public void configConstant(Constants me) {
    me.setError404View("/error/404.html");
    me.setError500View("/error/500.html");
}
```

- 目录位置

```bash
webapp/
├── error
│   ├── 404.html
│   └── 500.html
└── backend
    └── index.html
```
