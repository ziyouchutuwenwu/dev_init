# 环境配置

msys2 是 win 下模拟 linux 比较好的工具

## 安装 msys2

### 换源

```sh
sed -i "s#mirror.msys2.org/#mirrors.ustc.edu.cn/msys2/#g" /etc/pacman.d/mirrorlist*
```

### 安装必备工具

```sh
pacman -Syyu --noconfirm
pacman -S --noconfirm base-devel vim curl axel git
pacman -S --noconfirm python python-pip
```

### 创建脚本

```sh
vim /etc/profile.d/profile.sh
```

```sh
export MSYS2_ROOT=/c/msys64
export PATH=$MSYS2_ROOT/mingw64/bin:$PATH

alias pon="export http_proxy=socks5://10.0.2.1:1080 https_proxy=socks5://10.0.2.1:1080"
alias poff="unset http_proxy https_proxy"
```

```sh
source /etc/profile
```
