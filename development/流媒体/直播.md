# 直播

## hls

生成

```sh
ffmpeg -i 1.mp4 -c:v libx264 -c:a aac -hls_time 10 -hls_list_size 0 -hls_base_url ts/ -hls_segment_filename hls/ts/segment%03d.ts hls/playlist.m3u8
```

验证

```sh
ffplay hls/playlist.m3u8
```

## rtsp

生成

```sh
cvlc --loop "1.mp4" :sout="#rtp{sdp=rtsp://127.0.0.1:8554/demo}" :sout-keep
```

测试

```sh
ffplay rtsp://127.0.0.1:8554/demo
```
