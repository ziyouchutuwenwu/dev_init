# zellij

## 用法

### session 操作

查看

```sh
zellij ls
```

创建

```sh
zellij -s xxx
```

销毁

```sh
zellij k xxx
zellij ka
```

附加

```sh
zellij a xxx
```

### 快捷键

#### 锁定

锁定/解锁全局按键，防止冲突

```sh
ctrl+g
```

#### tab 操作

```sh
ctrl+t
```

#### panel 操作

同一个 tab 内的 panel 操作

```sh
ctrl+p
```

#### 修改大小

修改 panel 大小

```sh
ctrl+n
```

#### 切换 panel

```sh
ctrl+h
```

#### 其它功能

搜索/滚动/翻页

```sh
ctrl+s
```

#### session 相关

选择 session

退出而不销毁整个 session

```sh
ctrl+o
```

#### 退出

```sh
crtl+q
```
