# 备份

## 说明

官方版本 vscode 自带 github 同步

这里列出手动备份方式

## 手动

复制相关的 json

```sh
~/.config/$XXX/User/settings.json
~/.config/$XXX/User/keybindings.json
```

插件

```sh
# 备份
code --list-extensions > extensions.txt

# 安装
cat extensions.txt | xargs -L 1 code --install-extension

# 卸载多余
comm -23 <(code --list-extensions | sort) <(sort extensions.txt) | xargs -r -L 1 code --uninstall-extension

# 卸载所有
code --list-extensions | xargs -r -L 1 code --uninstall-extension
```
