# 导入导出

## 用法

导出

```sh
pg_dump -h 127.0.0.1 -p 5432 -U demo_user -d demo_db -f demo_db.sql

# 导出库 demo_db 的表 demo_table1
pg_dump -h 127.0.0.1 -p 5432 -U demo_user -t demo_table -d demo_db > demo_table.sql
```

导入

```sh
psql -h 127.0.0.1 -p 5432 -U demo_user -d demo_db -f demo_db.sql
psql -h 127.0.0.1 -p 5432 -U demo_user -d demo_db -f demo_table.sql
```
