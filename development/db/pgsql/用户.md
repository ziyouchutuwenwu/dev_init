# 用户

## 说明

进入 psql 的 shell

```sh
sudo su postgres
psql
```

## 用法

创建用户

```sql
create user demo_user with encrypted password '123456';
```

修改密码

```sql
-- 是在 psql 下， shell 下 不是这个命令
\password
\quit
```

显示用户

```sql
\du
```

删除用户

```sql
drop user demo_user;
```

指定用户登陆

```sh
psql -U demo_user -d demo_db
```

修改 db owner

```sql
alter database db_xxx owner to user_yyy;
```
