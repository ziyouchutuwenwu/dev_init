# 安装

## 基本配置

```sh
cd /etc/postgresql/${VERSION}/main
postgresql.conf
```

## 数据目录

默认位置

```sh
cat postgresql.conf | rg data_directory
```

或者

```sh
sudo su postgres
psql
```

```sql
-- psql模式下
show data_directory;
```

## 远程权限

postgresql.conf

```conf
listen_addresses ='*'
```

在 hba.conf 里面修改登录认证方式

```sh
cat postgresql.conf | rg hba_file
```

## 查看版本

```sql
select version();
```

## 重载配置

```sql
select pg_reload_conf();
```
