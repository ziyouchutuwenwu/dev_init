# 转发

## 说明

开启转发

```sh
左侧, 首页, 按 P 开启代理
```

## 配置

### codex

```sh
cc-switch --app codex
```

配置

```sh
api 地址
  https://api.deepseek.com/v1
路由
  打开
api key
  用 deepseek 的 api key
```

### claude

```sh
cc-switch --app claude
```

配置

```sh
api 地址
  https://api.deepseek.com/v1
api 格式
  OpenAI Chat Completions
api key
  用 deepseek 的 api key
```
