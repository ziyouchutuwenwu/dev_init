## 列表

### 技能

```sh
npx skills add https://github.com/vercel-labs/skills --skill find-skills
npx skills add https://github.com/anthropics/skills --skill skill-creator
```

```sh

```

[gstack](https://github.com/garrytan/gstack.git)

```sh

/design-xxx     设计相关
/review         代码审查
/qa             测试

```
