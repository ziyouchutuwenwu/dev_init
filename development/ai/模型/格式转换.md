# 格式转换

## 说明

支持 mirror 和 HTTP_PROXY

## 用法

安装

```sh
uv tool install 'optimum[onnx]'
```

在线转换

```sh
optimum-cli export onnx \
  --device cuda \
  --trust-remote-code \
  --model Qwen/Qwen2-VL-2B-Instruct \
  $DEST_DIR
```

离线转换

```sh
optimum-cli export onnx \
  --device cuda \
  --trust-remote-code \
  --model ./xxxx \
  $DEST_DIR
```
