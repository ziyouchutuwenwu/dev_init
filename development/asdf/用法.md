# 用法

## 下载

[地址](https://github.com/asdf-vm/asdf/releases)

## 例子

### 插件查询

```sh
asdf plugin list
asdf plugin list all
```

### 插件安装

```sh
asdf plugin add nodejs
asdf plugin remove nodejs
```

### vm 查看

查看已安装的

```sh
asdf list nodejs
```

查看版本号为 20 的所有版本

```sh
asdf list all nodejs 20
```

查看最新版

```sh
asdf latest nodejs

# 如果出来是 23.xxx, 希望获取之前版本的 lts
asdf latest nodejs 22
```

查看 bin 路径

```sh
asdf which node
```

查看 env

```sh
asdf env node
```

### vm 安装

```sh
asdf install nodejs 22.14.0
asdf uninstall nodejs 22.14.0
```

### 指定版本

```sh
# $HOME 下设置，类似全局
asdf set --home nodejs 22.14.0

# 从上一层目录开始找 .tool-versions 文件，如果没找到，就一直往上一层，到找到为止。
asdf set --parent nodejs 22.14.0
```

版本定义的文件

```sh
.tool-versions
```
