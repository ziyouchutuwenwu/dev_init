# 准备

## 说明

arm 不支持硬实时
x86 不支持设备树

所以用 arm64 测试

## 步骤

### 依赖库

uboot 和 linux 内核编译都需要

```sh
sudo apt install -y bison flex
```

### 编译环境

交叉编译器

```sh
# aarch64
sudo apt install gcc-aarch64-linux-gnu
```

~/.profile

```sh
export CROSS_COMPILE=aarch64-linux-gnu-
export ARCH=arm64
export KBUILD_OUTPUT=$(pwd)/output

export MAKEFLAGS="-j$(nproc)"
```
