# 配置

## 说明

推荐配置

## 步骤

```sh
apt install -y neovim
```

### profile

/etc/profile.d/path.sh

```sh
export PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin
```

~/.bashrc

```sh
[[ -f /etc/profile ]] && source /etc/profile
```

### 工具

```sh
source /etc/profile
apt install -y iputils-ping net-tools ncat file
```
