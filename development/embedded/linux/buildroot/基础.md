## 步骤

### 查看型号

搜索

```sh
# 位置在 configs，自己搜索真实板子型号
make list-defconfigs | rg mx6ull
```

### 准备

环境变量

```sh
export MAKEFLAGS="-j$(nproc) O=$(pwd)/build_output"
```

强烈建议参考 [配置分离](./分离.md) 做配置分离

### deconfig

```sh
make qemu_aarch64_virt_defconfig
```

### menuconfig

```sh
make menuconfig
```

```sh
Bootloaders  --->
  [*] U-Boot
  (这里配置路径) Board defconfig
  [*] U-Boot needs gnutls
  U-Boot binary format  --->
    [*] u-boot

Target packages  --->
  # 这个要选中，不然 vim 和 bash 都看不到
  [*] Show packages that are also provided by busybox
    Development tools  --->
      # gnu sed，自带的不标准，elixir 会报错
      [*] sed
      [*] binutils
      [*] binutils binaries
    # sh 对标准的支持太差
    Shell and utilities  --->
      [*] bash
    Text editors and viewers  --->
      [*] vim
```

### 编译

```sh
make
```
