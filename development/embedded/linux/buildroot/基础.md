## 配置

### deconfig

搜索

```sh
export MAKEFLAGS="-j$(nproc) O=$(pwd)/output"

# 位置在 configs，自己搜索真实板子型号
make list-defconfigs | rg mx6ull
```

生成默认配置

```sh
make qemu_aarch64_virt_defconfig
```

强烈建议参考 [配置保存](./配置保存.md) 做配置分离

### menuconfig

make menuconfig
