# 配置保存

## 说明

.config 文件里面包含所有的配置，BR2_DEFCONFIG 是 .config 里面，用户自定义的设置

配置以后，恢复以后，不需要再重新 menuconfig

## 分类

### uboot 配置

```sh
make menuconfig
```

设置路径

```sh
Bootloaders  --->
  U-boot  --->
    U-Boot configuration --->
      (X) Using a custom board (def)config file
    (board/demo_vendor/demo_board/uboot_defconfig) Configuration file path
```

文件占位

```sh
touch board/demo_vendor/demo_board/uboot_defconfig
```

详细配置

```sh
make uboot-menuconfig
```

保存

```sh
make uboot-update-defconfig
```

### 板级配置

这个最好最后一个配置，可以把别的配置都保存到一起

```sh
# 保存
make BR2_DEFCONFIG=board/demo_vendor/demo_board/demo_board.defconfig savedefconfig
```

还原配置

```sh
make BR2_DEFCONFIG=board/demo_vendor/demo_board/demo_board.defconfig defconfig
```
