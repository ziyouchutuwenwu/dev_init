# 配置保存

## 说明

配置以后，希望可以保存好，省的再来，很烦

## 例子

### 整体配置

保存配置

```sh
make O=output BR2_DEFCONFIG=board/demo_vendor/demo_board/demo_board.defconfig savedefconfig
```

还原配置

```sh
make O=output BR2_DEFCONFIG=board/demo_vendor/demo_board/demo_board.defconfig defconfig
```

### 内核配置

menuconfig

```sh
make O=output menuconfig
```

先写好内核配置文件的路径

```sh
Kernel  --->
  Kernel configuration ()  --->
    (X) Using a custom (def)config file
  (board/demo_vendor/demo_board/linux_defconfig) Configuration file path
```

准备一个空的，否则会报错

```sh
touch board/demo_vendor/demo_board/linux_defconfig
```

修改内核配置

```sh
make O=output linux-menuconfig
```

保存配置，保存到 `board/demo_vendor/demo_board/linux_defconfig`

```sh
make O=output linux-update-defconfig
```

不需要还原，make 的时候会自动读取自定义的配置
