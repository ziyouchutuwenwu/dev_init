# 打包

## 说明

多数板不需要自己写，会自动生成

自己画的板，需要写

## 步骤

配置

```sh
make menuconfig
```

```sh
Host utilities  --->
  [*] host dosfstools
  [*] host genimage
  [*] host mtools
```

```sh
System configuration  --->
  # 此时用来打包固件
  (board/demo_vendor/demo_board/post-image.sh) Custom scripts to run after creating filesystem images
```

post-image.sh

```sh
#!/bin/sh

BOARD_DIR="$(dirname "$0")"

# BINARIES_DIR 就是 output/images 目录
cp "${BOARD_DIR}/extlinux.conf" "${BINARIES_DIR}/"
cp "${BUILD_DIR}"/linux-*/arch/arm64/boot/dts/demo_board.dtb "${BINARIES_DIR}/"

support/scripts/genimage.sh -c "${BOARD_DIR}/genimage.cfg"
```

```sh
chmod a+x post-image.sh
```

extlinux.conf

```conf
LABEL linux
KERNEL /Image
FDT /demo_board.dtb
APPEND root=/dev/vda2 rootwait console=ttyAMA0
```

genimage.cfg

```cfg
image boot.vfat {
    vfat {
        file Image {
            image = "Image"
        }

        file demo_board.dtb {
            image = "demo_board.dtb"
        }

        file extlinux/extlinux.conf {
            image = "extlinux.conf"
        }
    }

    size = 64M
}

image disk.img {
    hdimage {
    }

    partition boot {
        partition-type = 0xC
        bootable = "true"
        image = "boot.vfat"
    }

    partition rootfs {
        partition-type = 0x83
        image = "rootfs.ext4"
    }
}
```
