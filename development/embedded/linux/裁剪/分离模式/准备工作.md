# 准备工作

## 依赖库

uboot 和 linux 内核编译都需要

```sh
sudo apt install -y bison flex
```

## 编译环境

交叉编译器

```sh
sudo apt install gcc-arm-linux-gnueabi
```

~/.profile

```sh
export CROSS_COMPILE=arm-linux-gnueabi-
export ARCH=arm

# uboot, linux 内核, busybox 用这个，buildroot 不支持这个
export KBUILD_OUTPUT=$(pwd)/output
```
