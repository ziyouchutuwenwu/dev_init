# 设备树

## 说明

加载方式

```sh
app.overlay
  任何开发板都能加载

boards/stm32f411ceu6.overlay
  west build -b stm32f411ceu6 时加载
  实际上是加载 repo/zephyr/boards/某个开发板下，dts 同名字的 overlay

xxx.overlay
  DDTC_OVERLAY_FILE 手动指定
```

## 例子

### 代码

boards/xxx.overlay

```c
/ {
    aliases {
      led0 = &led_label0;
    };

    // group 是给人看的
    leds {
      // 这个很重要，不能乱写
      compatible = "gpio-leds";

      // label: node_name
      led_label0: led_node0 {
        // 属性
        gpios = <&gpio9 0 GPIO_ACTIVE_HIGH>;
        label = "demo led";
      };
    };
};
```

引用

```c
#define LED0_NODE DT_ALIAS(led0)

static const struct gpio_dt_spec led = GPIO_DT_SPEC_GET(LED0_NODE, gpios);
```

### 编译

自动加载

```sh
west build -b stm32f411ceu6 -d debug
```

指定 overlay

```sh
west build -b stm32f411ceu6 -d debug -- -DDTC_OVERLAY_FILE=boards/aaa.overlay
```
