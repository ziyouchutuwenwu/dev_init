# 环境

## 说明

目录结构

```sh
embedded/zephyr
├── env.sh
├── sdk
├── repo
└── venv
```

## 步骤

准备

```sh
cmake wget ninja gperf
```

### venv

```sh
uv venv venv
cd venv
uv pip install west

# 后面会用到的库
uv pip install jsonschema pyelftools patool requests semver tqdm
```

### repo

```sh
cd repo

# 初始化主仓库
west init

# 下载各厂商驱动，支持断点续传
west update

# 全量下载
west update -f always
```

注册给 cmake

如果更换目录，则手动运行

```sh
west zephyr-export
```

### sdk

版本在 sdk/sdk_version

```sh
west sdk install --install-dir ~/dev/embedded/zephyr/sdk
```

如果它在 sdk 下创建了子目录，则手动挪到 sdk 里面

```sh
# 下载 toolchain，位置在 sdk/gnu
cd sdk
./setup.sh -t all
```

env.sh

```sh
#!/usr/bin/env bash

ZEPHYR_ROOT="$HOME/dev/embedded/zephyr"
WORKSPACE="$ZEPHYR_ROOT/repo"

ORIGINAL_DIR="$(pwd)"

cd "$WORKSPACE" || return

source "$ZEPHYR_ROOT/venv/bin/activate"
source zephyr/zephyr-env.sh

export ZEPHYR_TOOLCHAIN_VARIANT=zephyr
export ZEPHYR_SDK_INSTALL_DIR="$ZEPHYR_ROOT/sdk"

cd "$ORIGINAL_DIR"
```

### 测试
