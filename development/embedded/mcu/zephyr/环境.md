# 环境

## 说明

目录结构

```sh
zephyr
├── sdk
├── repo
└── venv
```

venv 用 uv 创建

```sh
uv venv venv
```

## 步骤

### 准备

工具

```sh
cmake wget ninja
```

### venv

```sh
cd venv
uv pip install west

# 后面会用到的库
uv pip install jsonschema pyelftools patool requests semver tqdm
```

### 源码

激活 python

```sh
source venv/bin/activate
```

```sh
cd repo

# 初始化主仓库
west init

# 下载各厂商驱动
# 断点续传
west update

# 全量下载
west update -f always

# 把 repo 和单元测试添加到 cmake 的包索引里面
# 更换目录后要运行
west zephyr-export
```

### sdk

```sh
west sdk install --install-dir ~/dev/zephyr/sdk
```

如果它在 sdk 下创建了子目录，则手动挪到 sdk 里面

```sh
# ~/dev/zephyr/repo/zephyr/sdk_version 确定版本

# 解压
cd sdk
./setup.sh -t all
```

env.sh

```sh
#!/usr/bin/env bash

ZEPHYR_ROOT="$HOME/dev/zephyr"
WORKSPACE="$ZEPHYR_ROOT/repo"

ORIGINAL_DIR="$(pwd)"

cd "$WORKSPACE" || return

source "$ZEPHYR_ROOT/venv/bin/activate"
source zephyr/zephyr-env.sh

export ZEPHYR_TOOLCHAIN_VARIANT=zephyr
export ZEPHYR_SDK_INSTALL_DIR="$ZEPHYR_ROOT/sdk"

cd "$ORIGINAL_DIR"
```

### 测试

source ~/dev/zephyr/env.sh
