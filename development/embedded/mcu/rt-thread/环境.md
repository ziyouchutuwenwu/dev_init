# 环境

## 说明

[官网](https://github.com/RT-Thread/rt-thread/releases)下载源码

## 步骤

下载 tar.gz，解压到 src

```sh
rtt
├── env.sh
├── src
├── toolchain
└── venv
```

```sh
uv venv venv
cd venv
uv pip install scons
uv pip install kconfiglib
uv pip install requests tqdm
```

```sh
# menuconfig 以后，会下载 env 软件包
scons --menuconfig
```

下载，解压到 toolchain

```sh
https://developer.arm.com/downloads/-/arm-gnu-toolchain-downloads
```

env.sh

```sh
#!/usr/bin/env bash

RTT_TOP="$HOME/dev/rtt"

# rtt 需要导出的变量
export RTT_ROOT="$RTT_TOP/src"
export RTT_CC="gcc"
export RTT_EXEC_PATH="$RTT_TOP/toolchain/bin"

export PATH="$RTT_EXEC_PATH:$PATH"

ORIGINAL_DIR="$(pwd)"

source "$RTT_TOP/venv/bin/activate"
source ~/.env/env.sh

cd "$ORIGINAL_DIR"
```

某个 bsp

```sh
# clean
scons --clean

# 编译
scons

# 为 vscode 打包
scons --target=vsc --dist
```
