# 设备树

## 说明

缩进用 tab, 不能用空格

## 文件名

```sh
boards
└── esp32s3_devkitc_procpu.overlay
```

```sh
west boards
esp32 的板子双核，west build 的时候会提示你，比如 esp32s3_devkitc/esp32s3/procpu
```

## 语法

节点

```c
// 节点名
node_name {
};

// label: 节点名
label: node_name {
};

// label: 引用其它节点的 label
label: &other_node_label {
};

// label: 节点名字@寄存器16进制地址
label: node_name@address {
};
```

compatible

```sh
# 固定的一些值，需要内核支持
compatible = "厂商,具体型号", "通用类型";
# 例子
compatible = "worldsemi,ws2812-i2s";
compatible = "espressif,esp32s3", "esp,esp32";
```

代码引用

```c
/ {
 // 代码内引用
 aliases {
  led0 = &led0;
 };
```
