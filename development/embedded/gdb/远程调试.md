# 远程调试

## 用法

### 服务端

编译好的 bin 复制到远程服务器，远程执行

```sh
gdbserver :12345 ./xxxxxx
```

正常输出如下，可能需要 root 权限

```sh
Listening on port 12345
```

### 调试端

vscode 的 launch.json

```json
{
  "type": "gdb",
  "request": "launch",
  "name": "remote_debug",
  "target": "${workspaceFolder}/output/main",
  "cwd": "${workspaceRoot}",
  // remote 后面是服务器的 ip 和 端口
  "autorun": ["target remote xx.xx.xx.xx:12345", "load ${workspaceRoot}/output/main"],
  "gdbpath": "/usr/bin/gdb-multiarch"
}
```
