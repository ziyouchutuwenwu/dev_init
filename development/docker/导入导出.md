# 导入导出

## 用法

### 容器 -> image

只有单层镜像

```sh
# 容器生成 tar
docker export ${CONTAINER_ID} > xxx.tar

# tar 保存为 image
docker import xxx.tar image_name:tag
```

方法 2

```sh
docker commit ${CONTAINER_ID} image_name:tag
```

### image

```sh
# 保存为 tar
docker save -o xxx.tar redis:5.0.0

# 这个 tar 和容器保存的 tar 不兼容
docker load -i xxx.tar
```
