# 权限报错

## 说明

| Group        | 用途说明                                            |
| ------------ | --------------------------------------------------- |
| libvirt      | `virt-manager` 管理虚拟机<br>`virsh` CLI 管理虚拟机 |
| libvirt-qemu | `libvirt` 启动 vm `qemu-system-*`                   |
| kvm          | 手动运行 `qemu-system-xxx` 用                       |

## 调试

查看 group 有没有

```sh
getent group kvm
getent group libvirt
getent group libvirt-qemu
```

## 配置

如果你的 vm 目录不是标准目录，建议

```sh
/etc/libvirt/qemu.conf
```

```conf
user = "xxx"
```
