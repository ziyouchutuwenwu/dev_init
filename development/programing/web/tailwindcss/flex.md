# flex

## 说明

有主轴和交叉轴的概念

## 主轴

| 项目 | flex-row       | flex-col |
| ---- | -------------- | -------- |
| 说明 | 横向主轴(默认) | 纵向主轴 |

## 对齐

| 项目             | justify-xxx     | items-xxx     |
| ---------------- | --------------- | ------------- |
| 说明             | 主轴上          | 交叉轴上      |
| 起始             | justify-start   | items-start   |
| 居中             | justify-center  | items-center  |
| 末尾             | justify-end     | items-end     |
| 等距（主轴独有） | justify-between | —             |
| 拉伸             | —               | items-stretch |
