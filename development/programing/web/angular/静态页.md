# 静态页

## 说明

```sh
ssr 需要 nodejs 运行
ssg 纯静态页，推荐
```

## 步骤

### 准备

已有项目添加 ssr

```sh
ng add @angular/ssr
```

### ssr

server 和 browser 都需要

```sh
server 存 ssr 相关
browser 存 assets 等
```

```sh
# 对于在哪里运行，没有要求，都可以
PORT=1234 node server/server.mjs
```

### ssg

angular.json

```sh
# 修改
"outputMode": "static"
```
