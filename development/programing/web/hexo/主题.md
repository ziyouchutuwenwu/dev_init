## 配置

安装插件

```sh
# 未来不需要任何修改
git submodule add https://github.com/CodeDaraW/Hacker themes/Hacker

# 如果你要自己修改，比如加广告之类，不要用 submodule
git clone --depth 1 https://github.com/CodeDaraW/Hacker.git
rm -rf Hacker/.git
```

生成配置

```sh
cp -rf themes/Hacker/_config.example.yml _config.Hacker.yml
```

\_config.Hacker.yml

```yaml
menu:
  首页: /
  归档: /archives
  标签: /tags
  关于: /about
```

\_config.yml

```yaml
theme: Hacker
```

source/tags/index.md

```markdown
---
title: tags
date: 2025-10-24 19:46:11
type: "tags"
layout: tags
---
```
