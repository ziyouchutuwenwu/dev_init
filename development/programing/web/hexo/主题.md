## 配置

安装插件

```sh
git clone https://github.com/CodeDaraW/Hacker themes/Hacker
rm -rf themes/Hacker/.git
```

生成配置

```sh
cp -rf themes/Hacker/_config.example.yml _config.Hacker.yml
```

\_config.Hacker.yml

```yaml
menu:
  首页: /
  归档: /archives
```

\_config.yml

```yaml
theme: Hacker
```
