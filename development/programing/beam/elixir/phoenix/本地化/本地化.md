# 本地化

## 步骤

### 目录准备

准备好目录即可

```sh
priv/gettext/zh_CN/LC_MESSAGES/
priv/gettext/en/LC_MESSAGES/
```

### 合并

合并以后，在 LC_MESSAGES 目录下会生成 default.po 文件

如果更新，重新执行命令即可

```sh
mix gettext.extract --merge
```

msgid 就是 key

```sh
msgid "aaa"
msgstr "english aaa"
```

## 设置

### 方法 1

config.exs

```elixir
config :web_demo, WebDemoWeb.Gettext,
  default_locale: "zh_CN"
```

### 方法 2

plug 里面

```elixir
defmodule WebDemoWeb.Plugs.Locale do
  require Logger
  use Gettext, backend: WebDemoWeb.Gettext

  def init(default_locale) do
    Gettext.put_locale("zh_CN")

    # 返回值是 call 的第二个参数
    default_locale
  end

  # 必须返回 conn
  def call(conn, _default_locale) do
    locale = Gettext.get_locale()
    Logger.debug(locale)

    conn
  end
end
```

heex 里面

```elixir
<p><%= gettext("aaa") %></p>
```
