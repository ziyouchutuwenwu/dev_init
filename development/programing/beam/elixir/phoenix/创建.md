# 创建

## 准备

```sh
mix archive.install hex phx_new
```

指定版本

```sh
mix archive.install hex phx_new 1.8.0-rc.0
```

## 用法

基本用法

```sh
mix phx.new web_demo
```

其它参数

```sh
--no-dashboard \
--no-gettext \
--no-mailer \
\
--database mysql \
--no-ecto \
\
--no-live \
--no-assets \
--no-html
```

## 启动

```sh
mix phx.server
```

调试

```sh
iex -S mix phx.server
```
