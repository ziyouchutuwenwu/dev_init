# 模板注入

## 说明

root.html.heex 里面添加动态标签，controller 里面注入，esbuid 配置编译

## 例子

router.ex

```elixir
scope "/", WebDemoWeb do
  pipe_through :browser

  get "/aaa", PageController, :aaa
  get "/bbb", PageController, :bbb
end
```

root.html.heex

```html
<%= if assigns[:page_js] do %>
  <script
    defer
    phx-track-static
    type="text/javascript"
    src={"/assets/js/pages/#{assigns[:page_js]}.js"}
  >
  </script>
<% end %>
```

page_controller.ex

```elixir
defmodule WebDemoWeb.PageController do
  use WebDemoWeb, :controller

  def aaa(conn, _params) do
    render(conn, :aaa, page_js: "aaa")
  end

  def bbb(conn, _params) do
    render(conn, :bbb, page_js: "bbb")
  end
end
```

aaa.html.heex

```html
<Layouts.app flash="{@flash}">
  <div class="flex items-center justify-center min-h-[60vh]">
    <button
      type="button"
      onclick="demo1.on_click()"
      class="rounded-lg bg-blue-600 px-6 py-3 text-white font-semibold shadow-md hover:bg-blue-700 transition-colors duration-200">
      click aaa
    </button>
  </div>
</Layouts.app>
```

bbb.html.heex

```html
<Layouts.app flash="{@flash}">
  <div class="flex items-center justify-center min-h-[60vh]">
    <button
      type="button"
      onclick="demo2.on_click()"
      class="rounded-lg bg-green-600 px-6 py-3 text-white font-semibold shadow-md hover:bg-green-700 transition-colors duration-200">
      click bbb
    </button>
  </div>
</Layouts.app>
```

### js 代码

aaa.ts

```typescript
interface Window {
  demo1: DemoClass;
}

class DemoClass {
  on_load(): void {
    console.log("on_load in DemoClass");
  }

  on_click(): void {
    console.log("on_click in DemoClass");
  }
}

window.demo1 = new DemoClass();
```

bbb.ts

```typescript
interface Window {
  demo2: {
    on_load: () => void;
    on_click: () => void;
  };
}

function on_load(): void {
  console.log("on_load in bb");
}

function on_click(): void {
  console.log("on_click in bb");
}

window.demo2 = { on_load, on_click };
```

config.exs

```elixir
config :esbuild,
  extra: [
    args:
      Path.wildcard("js/pages/**/*.ts") ++
        ~w(--bundle --loader:.js=ts --target=es2022 --outbase=js/pages
           --outdir=../priv/static/assets/js/pages
           --external:/fonts/* --external:/images/* --alias:@=.),
    cd: Path.expand("../assets", __DIR__),
    env: %{"NODE_PATH" => [Path.expand("../deps", __DIR__), Mix.Project.build_path()]}
  ]
```
