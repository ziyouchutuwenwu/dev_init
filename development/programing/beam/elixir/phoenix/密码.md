# 密码

## 说明

bcrypt 和 argon2 在调试的时候都不能断点

## 例子

### 创建项目

```sh
mix new demo
```

### deps

```elixir
{:pbkdf2_elixir, "~> 2.0"}
```

### 代码

```elixir
defmodule Demo do
  require Logger

  def demo do
    pwd = "123456"
    hashed_pwd = Pbkdf2.hash_pwd_salt(pwd)
    Logger.debug("#{inspect(hashed_pwd)}")

    Pbkdf2.verify_pass(pwd, hashed_pwd)
  end
end
```
