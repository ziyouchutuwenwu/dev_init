# 结构体

## 说明

类似于其他语言里面的对象属性

## 例子

defstruct 定义的结构体名字默认等于当前模块名（**MODULE**）

```elixir
defmodule Person do
  defstruct [:name, :age]

  def new(name, age) do
    # __MODULE__ 必须写，否则就是传统的 map 了
    %__MODULE__{name: name, age: age}
  end
end

person = Person.new("name1", 111)
person.name
person.age
```
