# 交叉编译

## 说明

elixir 项目交叉编译到开发板，c 库使用 musl 静态链接，支持 heartbeat

- 主机: Linux x86_64
- 需求: 用 `arm-linux-musleabihf` 静态库来跑，避开 uclibc
- 工具链: 因为要避开 uclibc， 所以用 arm-linux-musleabihf

## 步骤

下载 toolchain

```bash
https://musl.cc/arm-linux-musleabihf-cross.tgz
```

下载 erlang 源码

```bash
git clone --depth 1 --branch OTP-29.0.3 https://github.com/erlang/otp.git otp_src
```

erl-xcomp-arm-linux-musleabihf.conf

```conf
erl_xcomp_build=guess
erl_xcomp_host=arm-linux-musleabihf
erl_xcomp_configure_flags="--without-termcap --without-javac --without-ssl"

CC=arm-linux-musleabihf-gcc
# 不加 `-static`，避免影响 NIF 共享库
CFLAGS="-Os -Wall -g"
STATIC_CFLAGS="-static"
CPP=arm-linux-musleabihf-cpp
CXX=arm-linux-musleabihf-g++
CXXFLAGS="-Os -Wall"
LD=arm-linux-musleabihf-gcc
LDFLAGS="-static"
LIBS=

RANLIB=arm-linux-musleabihf-ranlib
AR=arm-linux-musleabihf-ar

erl_xcomp_after_morecore_hook=no
erl_xcomp_bigendian=no
erl_xcomp_clock_gettime_cpu_time=yes
erl_xcomp_getaddrinfo=yes
erl_xcomp_gethrvtime_procfs_self=no
erl_xcomp_dlsym_brk_wrappers=no
erl_xcomp_kqueue=no
erl_xcomp_linux_clock_gettime_correction=yes
erl_xcomp_linux_clock_gettime_correction_hz_1000=yes
erl_xcomp_linux_usable_clock_gettime=yes
erl_xcomp_linux_usable_sigev_thread_id=yes
erl_xcomp_poll=yes
erl_xcomp_posix_memalign=yes
erl_xcomp_pwritev=yes
erl_xcomp_readv=no
erl_xcomp_sendfile=yes
erl_xcomp_struct_flock=yes
erl_xcomp_sys_timezone=yes
erl_xcomp_timezone=yes
erl_xcomp_writev=yes

## -- pthreads --
erl_xcomp_pthread_getcpuclockid=yes
```

configure

```sh
export BASE_ROOT=$(pwd)
export TOOLCHAIN_ROOT=$BASE_ROOT/arm-linux-musleabihf
export PATH=$TOOLCHAIN_ROOT/bin:$PATH

cd otp_src
export ERL_TOP=$(pwd)
./otp_build configure --xcomp-conf=../erl-xcomp-arm-linux-musleabihf.conf
./otp_build boot -a
./otp_build release -a $BASE_ROOT/beam_erts
```

mix.exs

```elixir
def project do
  [
    .........
    releases: releases()
  ]
end

# 设置 ERTS_PATH，则交叉编译，否则本地编译
defp releases do
  [
    demo:
      [steps: [:assemble, :tar]]
      |> then(fn opts ->
        if erts = System.get_env("ERTS_PATH") do
          Keyword.put(opts, :include_erts, erts)
        else
          opts
        end
      end)
  ]
end
```

打包

```bash
cd demo
export ERTS_PATH=$BASE_ROOT/beam_erts/erts-17.0.3
MIX_ENV=prod mix release
```

减肥，可选

```bash
cd demo/_build/prod/rel/demo/erts-17.0.3/bin
arm-linux-musleabihf-strip beam.smp heart epmd erlexec run_erl erl_child_setup escript erlc
```

测试

```sh
mkdir demo
gunzip -c ./demo-0.1.0.tar.gz | tar xf - -C ./demo
```
