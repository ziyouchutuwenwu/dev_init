## 说明

## 步骤

推荐的目录结构

编译以后，rel 目录下，对应具体版本号

```sh
/opt/demo/
  ├── 0.1.0/
  │   ├── bin/demo
  │   ├── lib/
  │   └── releases/
  └── 0.2.0/
      ├── bin/demo
      ├── lib/
      └── releases/
```

代码

```elixir
defmodule Release.Upgrader do

  def to(version) do
    with {:ok, cmd} <- build_cmd(version) do
      :heart.set_cmd(cmd)
      System.halt()
    end
  end

  def build_cmd(version) do
    root = System.get_env("RELEASE_ROOT")
    name = System.get_env("RELEASE_NAME")

    if root && name do
      target = Path.join([Path.dirname(root), version, "bin", name])

      if File.exists?(target) do
        {:ok, ~c"#{target} daemon"}
      else
        {:error, {:target_not_found, target}}
      end
    else
      {:error, :missing_env}
    end
  end
end
```
