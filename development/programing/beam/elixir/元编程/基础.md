# 基础

## 例子

### 普通例子

```elixir
defmodule DemoMacro do
  # defmacro 把 plus 注册为宏
  defmacro plus(ast_x, ast_y) do
    quote do
      unquote(ast_x) + unquote(ast_y)
    end
  end
end
```

测试

```elixir
require DemoMacro
# 编译器看到，就会去调用 DemoMacro.plus 这个宏
DemoMacro.plus(11,22)
```

### bind_quoted

bind_quoted 绑定的值会保持不变

代码如下

```elixir
defmodule DemoMacro do
  require Logger

  defmacro double_puts(expr) do
    quote do
      Logger.debug(unquote(expr))
      Logger.debug(unquote(expr))
    end
  end
end
```

```elixir
defmodule DemoMacro do
  require Logger

  defmacro double_puts(expr) do
    quote bind_quoted: [aaa: expr] do
      Logger.debug(aaa)
    end
  end
end
```

测试

```elixir
require DemoMacro
DemoMacro.double_puts(:os.system_time)
```
