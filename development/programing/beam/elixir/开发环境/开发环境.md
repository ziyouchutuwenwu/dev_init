# 开发环境

## 用法

先设置镜像，类似

```sh
export HEX_UNSAFE_HTTPS=1
export HEX_MIRROR="https://hexpm.upyun.com"
```

安装, 可以使用 proxychains

```sh
mix local.hex
```

如果失败, 手动下载

```sh
https://hexpm.upyun.com/installs/1.14.0/hex-2.0.6.ez
https://repo.hex.pm/installs/1.14.0/hex-2.0.6.ez
```

然后执行

```sh
mix archive.install ./hex-2.0.6.ez
```

确认

```sh
mix hex.info
```
