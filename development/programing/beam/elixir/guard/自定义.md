# 自定义

## 说明

guard 跟在 when 后面

## 例子

```elixir
defmodule DemoGuard do
  defguard is_positive(n) when is_integer(n) and n > 0
end

defmodule Demo do
  import DemoGuard

   def check(n) when is_positive(n) do
    "#{n} 是正数"
  end

  def check(n) do
    "#{n} 不是正数"
  end
end
```

测试

```elixir
Demo.check(10)
Demo.check(0)
Demo.check(-5)
```
