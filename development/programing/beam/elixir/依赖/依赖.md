# 依赖

## 用法

加载本地库

```elixir
{:xxx, path: "../xxx"}
```

指定分支

```elixir
{:bumblebee, github: "elixir-nx/bumblebee", branch: "jk-http-proxy"}
```

指定分支

```elixir
{:bumblebee, github: "elixir-nx/bumblebee", branch: "jk-http-proxy"}
```

指定环境下加载

```elixir
{:exla, "~> 0.7", only: [:dev, :test]}
```

运行时并不加载

```elixir
{:ex_doc, "~> 0.29", only: :dev, runtime: false}
```
