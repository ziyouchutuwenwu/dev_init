# 自定义构建

## 说明

自定义 build.zig

## 步骤

```elixir
defmodule DemoZig do
  use Zig,
    otp_app: :demo,
    leak_check: true,
    dump_build_zig: true

  ~Z"""
  """
end
```

```sh
mix compile
```

然后把 build.zig 和 build.zig.zon 复制到 `zigler/build`

### 4. 启用

`lib/demo_zig.ex`

