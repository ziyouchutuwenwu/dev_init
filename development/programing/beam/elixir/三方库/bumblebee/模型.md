# 模型

## 用法

### 缓存目录

查看

```elixir
Bumblebee.cache_dir()
```

自定义

```sh
export BUMBLEBEE_CACHE_DIR=./models
```

### hf 模型

```elixir
# Bumblebee.load_model({:hf, "google-bert/bert-base-uncased"})
Bumblebee.load_model({:hf, "username/model-name"})
```

加载本地的 hf 的 cache，不访问 hf

```sh
export BUMBLEBEE_OFFLINE=true
```

### 本地模型

加载本地模型

```sh
models
└── my_bert
    ├── config.json
    ├── model.safetensors
    ├── tokenizer_config.json
    └── tokenizer.json
```

```elixir
Bumblebee.load_model({:local, "models/my_bert"})
Bumblebee.load_tokenizer({:local, "models/my_bert"})
```

### 能否加载

判断能不能加载, load_spec 很快

```elixir
# hf 的模型
Bumblebee.load_spec({:hf, "google-bert/bert-base-uncased"})
```

```sh
# 本地模型
Bumblebee.load_spec({:local, "models/my_bert"})
```
