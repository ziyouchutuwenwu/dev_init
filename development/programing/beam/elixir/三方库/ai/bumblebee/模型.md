# 模型

## 说明

一种是 hugging face 的模型缓存，一种是本地模型

## 用法

### 模型缓存

直接加载 hf 的模型，实际上是下载到缓存里面

```elixir
# {:hf, "google-bert/bert-base-uncased"})
Bumblebee.load_model({:hf, "username/model-name"})
```

查看缓存位置

```elixir
Bumblebee.cache_dir()
```

自定义缓存位置

```sh
export BUMBLEBEE_CACHE_DIR=./models
```

加载本地的 hf 缓存，不访问 hf

```sh
export BUMBLEBEE_OFFLINE=true
```

### 本地模型

加载本地模型

支持 pytorch_model.bin 和 model.safetensors

生产推荐 safetensors

```sh
models
└── my_bert
    ├── config.json
    ├── model.safetensors
    ├── tokenizer_config.json
    └── tokenizer.json
```

```elixir
Bumblebee.load_model({:local, "models/my_bert"})
Bumblebee.load_tokenizer({:local, "models/my_bert"})
```

### 能否加载

判断能不能加载, 用 load_spec 最快

```elixir
# hf 的模型
Bumblebee.load_spec({:hf, "google-bert/bert-base-uncased"})
```

```sh
# 本地模型
Bumblebee.load_spec({:local, "models/my_bert"})
```
