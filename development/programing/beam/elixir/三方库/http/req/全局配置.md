# 全局配置

## 说明

req 基于 finch, 全局配置就是配置 finch

## 例子

```elixir
defmodule Demo.Application do
  require Logger
  use Application

  def start(_type, _args) do
    Logger.debug("in app start")

    children = [
      {
        Finch,
        name: ConfigedFinch
      }
    ]

    opts = [strategy: :one_for_one, name: Demo.Supervisor]
    Supervisor.start_link(children, opts)
  end
end
```
