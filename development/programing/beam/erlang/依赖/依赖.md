# 依赖

## 说明

不能加载 elixir 的库

## 用法

### 远程依赖

hex.pm

```erlang
{deps,[
  {jsx, "3.1.0"}
]}.
```

github

```erlang
{deps, [
  {ets_cache, ".*", {git, "https://github.com/roowe/ets_cache", "master"}},
  {erlbus, {git, "https://github.com/cabol/erlbus", {tag, "v0.3.0"}}}
]}.
```

### 本地依赖

```erlang
{deps, [
  {demo_lib, {path, "lib/demo_lib"}}
]}.
```
