# 节点名

## 说明

sname 和 name

## 分类

sname

- 后面是主机名。
- 主机名不写，则自动用 hostname 代替
- 主机名不允许带点!!!!
- connect_node 的时候，必须带上主机名
- 主机名，不允许使用 ip

```erlang
erl -sname debug -setcookie 123456
erl -sname debug@manjaro -setcookie 123456
```

name

- 后面是域名。
- 不写域名，无法启动。
- 域名，带至少一个点!!!!
- 域名部分，允许使用 ip

```erlang
erl -name aaa@127.0.0.1 -setcookie 123456
erl -name aaa@manjaro.com -setcookie 123456
```

## 动态设置

```erlang
% 这是 longnames
net_kernel:start(['aaa@manjaro']).
net_kernel:longnames().

net_kernel:start(['debug@manjaro', shortnames]).
net_kernel:start(['debug@aaa.local', longnames]).
```

```erlang

```

## 连接

