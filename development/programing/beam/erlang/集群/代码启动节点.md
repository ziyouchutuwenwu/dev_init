# 代码启动节点

## 说明

在节点上启动节点

## 代码

longname

```sh
erl -name aaa@127.0.0.1 -setcookie 123456
```

```erlang
{ok, _Pid, PeerNode} = peer:start(#{
    name => bbb,
    host => "127.0.0.1",
    cookie => <<"123456">>,
    args => ["-setcookie", "123456"]
}).
```

shortname

```sh
erl -sname aaa -setcookie 123456
```

```erlang
{ok, _Pid, PeerNode} = peer:start(#{
    name => bbb,
    % 可以不写
    % host => "localhost",
    cookie => <<"123456">>,
    args => ["-setcookie", "123456"]
}).
```
