# 通信端口

## 说明

在 min 和 max 之间选一个可用的，默认选最小的

```erlang
inet_dist_listen_min
inet_dist_listen_max
```

## 配置

### 命令行

```sh
erl -name aaa@127.0.0.1 -setcookie 111 -kernel inet_dist_listen_min 4111 inet_dist_listen_max 4111
erl -name bbb@127.0.0.1 -setcookie 111 -kernel inet_dist_listen_min 4222 inet_dist_listen_max 4222
erl -name ccc@127.0.0.1 -setcookie 111 -kernel inet_dist_listen_min 4333 inet_dist_listen_max 4333
```

### rebar3

vm.args

```erlang
-kernel inet_dist_listen_min 9100
-kernel inet_dist_listen_max 9105
```

### 动态配置

```erlang
application:set_env(kernel, inet_dist_listen_min, 9100).
application:set_env(kernel, inet_dist_listen_max, 9105).
```
