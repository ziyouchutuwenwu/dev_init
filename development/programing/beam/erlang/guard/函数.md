# 函数

## 说明

guard 跟在 when 后面

## 例子

```erlang
-module(demo).

-export([demo/1]).

demo(X) when is_integer(X), X > 0 ->
    io:format("正整数: ~p~n", [X]);
demo(X) when is_integer(X) ->
    io:format("非正整数: ~p~n", [X]);
demo(_) ->
    io:format("不是整数~n").
```

测试

```erlang
demo:demo(10).
demo:demo(-5).
demo:demo("abc").
```
