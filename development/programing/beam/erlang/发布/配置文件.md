# 配置文件

## 注意

最外层的 aaaaaa 为 src/xxxx.app.src 里面的 application

### 例子

rebar.config

```erlang
{sys_config, "config/sys.config"}.

% 支持环境变量， 编译以后，会自动生成 sys.config
{sys_config_src, "config/sys.config.src"}
```

sys.config

```erlang
[
  {aaaaaa,[
    {key1, aaa},
    {key2, bbb},
    {key3, ccc}
  ]}
].
```

sys.config.src

```erlang
[
  {aaaaaa,[
    {key1, "${AAA}"},
    {key2, "${BBB}"},
    {key3, ccc}
  ]}
].
```

### 获取数据

```erlang
application:get_all_env(aaaaaa).
application:get_env(aaaaaa, key3).
```
