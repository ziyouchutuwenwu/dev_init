# 参数

## 说明

src 后缀支持环境变量

## 位置

```sh
config
├── sys.config.src
└── vm.args.src
```

## 分类

### sys.config.src

见配置文件的文档

### vm.args.src

配置虚拟机参数，运行时加载环境变量

```erlang
-kernel inet_dist_listen_min ${DIST_PORT}
-kernel inet_dist_listen_max ${DIST_PORT}
```

测试

```sh
rebar3 as prod release
DIST_PORT=6666 _build/prod/rel/demo/bin/demo console
```

vm 参数见 [这里](https://www.erlang.org/doc/man/erl.html)
