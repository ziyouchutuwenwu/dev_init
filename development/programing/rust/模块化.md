# 模块化

## 说明

so 里面代码模块化

exe 里面调用

## 例子

### lib

```sh
cargo new demo_lib --lib
```

目录结构

```sh
src
├── lib.rs
└── mod1
    ├── aaa.rs
    └── mod.rs
```

文件

lib.rs

```rust
pub mod mod1;
```

mod.rs

```rust
pub mod aaa;
```

aaa.rs

```rust
pub fn demo(left: usize, right: usize) -> usize {
    left + right
}
```

### 调用

Cargo.toml

```toml
[dependencies]
demo_lib = { path = "../demo_lib" }
```

main.rs

```rust
use demo_lib::mod1::aaa;

fn main() {
    let aaa = demo_lib::mod1::aaa::demo(11, 33);
    let bbb = aaa::demo(22, 33);
    println!("result is {} {}", aaa, bbb);
}
```
