# 项目级

## 用法

### 创建

默认会生成 `.venv` 目录

```sh
uv init demo
```

指定版本

```sh
uv init demo --python=3.10
```

### 依赖

增加依赖

```sh
# 这个会更新 pyproject.toml
uv add requests
```

```sh
# 把 pyproject.toml 内的包更新到 requirements.txt
uv pip compile pyproject.toml -o requirements.txt
```

同步依赖

```sh
uv sync
uv pip sync ./pyproject.toml
```

### 运行

基于当前 venv 运行

```sh
uv run xxx
uv run main.py
```
