# 开发环境

## 说明

### 终端配置

目录结构

```sh
./ziglang
├── zig
│   ├── lib
│   └── zig
└── zls
```

### 环境变量

```sh
export ZIGROOT=$HOME/dev/ziglang
export PATH=$PATH:$ZIGROOT:$ZIGROOT/zig
```

### vscode 配置

.vscode/settings.json

```json
{
  "zig.zls.zigLibPath": "/home/mmc/dev/ziglang/zig/lib/",
  "zig.path": "/home/mmc/dev/ziglang/zig/zig",
  "zig.zls.path": "/home/mmc/dev/ziglang/zls"
}
```
