# 动态库

## 例子

build.zig

```zig
const lib = b.addLibrary(.{
    // .dynamic 或者 .static
    .linkage = .dynamic,
    .name = "zig_demo",
    .root_module = b.createModule(.{
        .root_source_file = b.path("src/root.zig"),
        .target = target,
        .optimize = optimize,
    }),
});
b.installArtifact(lib);
```
