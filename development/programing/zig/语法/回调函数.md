# 回调函数

## 例子

```zig
const std = @import("std");

const CallBackProc = fn (arg: u64) void;

fn demo(callback_proc: CallBackProc, ms: u64) void {
    callback_proc(ms);
}

fn on_callback(argument: u64) void {
    std.log.debug("proc args {d}", .{argument});
}

pub fn main() void {
    demo(on_callback, 10000);
}
```
