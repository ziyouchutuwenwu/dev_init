## 配置

### 准备工作

先开启 ssh 的 root 登录，然后 scp 过去运行

### 配置代理

带桌面的话，配置普通用户的 git 代理

```sh
vi ~/.gitconfig
```

```sh
[http]
  proxy = socks5://127.0.0.1:1080
[https]
  proxy = socks5://127.0.0.1:1080
```

### 脚本结束

#### 分辨率调整

qemu 下，如果显示不全，可以手动调节分辨率

```sh
设置 -> 设置管理器 -> 显示
```

分辨率改成

```sh
1280x720* 16:9
```

#### 调整时间显示格式

桌面右上角设置

```sh
时间 -> 右键 -> 属性
  外观 -> 布局 -> 数字式
  时钟选项 -> 布局 -> 仅时间
  时间 -> 格式 -> 自定义格式 -> %Y-%m-%d %H:%M:%S
```
