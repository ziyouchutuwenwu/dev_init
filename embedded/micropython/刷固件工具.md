# 刷固件用具用法

## 先进入 dfu 模式，两种方式

- 硬件通过短接 boot0 和 3.3v
- 或者

```python
import pyb
pyb.bootloader()
```

## micrppython 官网下载刷固件工具，编译

- 查看 dfu 设备

```bash
sudo ./dfu-util -l
```

- 刷固件

```bash
sudo ./dfu-util --alt 0 -D /home/mmc/dev/embedded/叁议小店PYBOARD开发板配套资料/08_pyboard固件/pybv11-20190201-v1.10-45-g67689bfd7.dfu
```

### 如果刷错启动不了，需要硬件重新进入 dfu，再刷
