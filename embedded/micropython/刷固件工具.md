### 先进入dfu模式，两种方式
- 硬件通过短接boot0和3.3v
- 或者
```
import pyb
pyb.bootloader()
```

### micrppython官网下载刷固件工具，编译
- 查看dfu设备
```
sudo ./dfu-util -l
```

- 刷固件
```
sudo ./dfu-util --alt 0 -D /home/mmc/dev/embedded/叁议小店PYBOARD开发板配套资料/08_pyboard固件/pybv11-20190201-v1.10-45-g67689bfd7.dfu
```

### 如果刷错启动不了，需要硬件重新进入dfu，再刷
