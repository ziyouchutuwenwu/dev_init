# 以 micropython_cn 的 nano 为例

- 下载源码

```bash
git clone https://github.com/micropython/micropython
git clone https://github.com/micropython-Chinese-Community/MicroPython_firmware micropython_firmware_cn
```

- 解压 PYBNanoV2/board_define.zip，复制到 micropython/ports/stm32/boards/

- 在代码根目录

```bash
git submodule update --init
```

- 编译 py 的交叉编译器，mpy-cross

```bash
make -C mpy-cross
make -C ports/stm32 BOARD=PYBNANO_V2
```

- 如果需要使用双精度浮点功能，可以这样

```bash
make -C ports/stm32 BOARD=PYBNANO_V2 MICROPY_FLOAT_IMPL=double
```

- 如果出现错误，可以 clean 一下再重新 build

```bash
make -C ports/stm32 BOARD=PYBNANO_V2 clean
```

- 编译成功以后，板子进入 dfu 模式，烧写固件
