# 西门子 plc

## fc 和 fb

- fc 相当于普通函数调用，不支持静态变量
- fb 相当于支持静态变量的函数调用，主调 db 的部分，一般会提示添加一个 db，这个 db 即背景数据块。

## fb 和 db

### 有三种方式做关联

- 单对单，fb 自己的数据存自己的 db
- 多重实例模式

```txt
例如，主 fb 引用子 fb，主 fb 有自己的 db；那么，在主 fb 里面，拖子 fb 进去，并声明为多重实例。
实际上是在主fb的变量区，把子 fb 作为 static 变量定义了。
而子 fb 没有自己的 db，使用的是主 fb 的 db；实际上是在主fb的db里面，可以看到子fb的定义
如果在子fb里面定义了一系列的测试变量，在主fb的db里面也都能看到，类似指针引入

```

- 参数模式，和多重实例差不多，不过是以参数的模式存在主 fb 的变量区
