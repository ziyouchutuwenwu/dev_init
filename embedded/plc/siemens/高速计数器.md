# 高速计数器

## 介绍

### 场景

用于计算脉冲数或者行走的距离

### 有以下功能

```bash
复位
硬件门（启动）
脉冲捕获
比较输出
```

## 计数模式

### 单相

```bash
内部方向控制单相计数：一路输入
外部方向控制单相计数：两路输入
```

### 两相位

增减计数：两路输入

### A/B 计数器

AB 相交计数：两路输入

### AB 计数器四倍频

AB 相交 4 倍频：两路输入

## 操作步骤

### 组态

设备组态，高速计数器，

```bash
功能，根据上面的 4 种类型选择
初始值，参考值用于输出比较或者中断事件。初始参考值是，初始参考值2是
同步输入，相当于于复位信号
捕捉输入，即输入捕获
门输入，即额外的启动信号
比较输出，当某个值等于参考值的时候，输出脉冲
组态事件，根据具体的事件，可以配置对应的中断，在中断里面写代码
```

### 注意

把 DI 里面的滤波需要小于频率对应的事件，否则脉冲会被滤波过滤掉
