# pid 控制

主要用于温度等恒定控制

## 组态

### 输入输出说明

#### 输入

传统 input 为浮点数, PER 为模拟量,一般选择浮点输入

#### 输出

看需求,如果是固态继电器控制通断,则用 pwm 模式的开关波形输出；
如果控制设备需要模拟量,则选择 PER 输出

### pid 参数

比例增益用于控制是否快速增大功率
积分作用时间越大越平缓
微分时间越大,抖动越大
记住这里你设置的采样算法时间

## 开发

在程序块里面创建 `cyclic interrupt`,周期为刚设置的采样算法时间
在里面调用 PID_Compact 方法.
注意,db 选择刚创建的工艺对象里面的 db,否则就是重新创建一个 pid

## 调试

- pid 参数部分,可以跳转到组态页面修改参数
- 改完以后,在`上传 pid 参数`部分,有个感叹号,可以更新状态
- 调节可以手动, 可以自动, 可以结合着来

### 手动

自己手动调节,主要是 `比例增益`, `积分作用时间`, `微分作用时间`

### 自动

在调节模式下的 `预调节` 上选择 start 即可
