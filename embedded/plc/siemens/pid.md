# pid

## 介绍

### 比例(P)

是按比例反应系统的偏差，系统一旦出现了偏差，比例调节立即产生调节作用用以减少偏差。比例作用大，可以加快调节，减少误差，但是过大的比例，使系统的稳定性下降，甚至造成系统的不稳定。

### 积分(I)

消除误差, 可以理解为多次求平均

### 微分(D)

相当于加入了阻尼

## 步骤

先组态, 再做函数调用

## 注意

```bash
Pid_Compact 的调用必须放在 ob30 的循环中间里面, 循环中断时间为 pid 的采样时间
使用 Pid_Compact 指令的时候, 注意选择已经组态好的对象的 db
```

## 说明

### 组态说明

组态的时候, `过程值限制` 和 `输出值限制` 都需要做

#### 输入

```bash
SetPoint为设定值, 比如80度
Input 用于工程量输入
Input_PER 用于模拟量输入(因为模拟量输入对应实际的百分比),如果启用Input_PER, 则需要在组态里面做`过程值标定`
```

### 输出

```bash
Output 用于工程量输出
Output_PER 用于模拟量输出
Output_PWM 用于继电器/开关量做一定频率的通断输出
```

## 调试

### 自动调试

```bash
在创建的工艺对象下,选择 pid 的调试
点击采样时间的 'start' 按钮
点击底部的 'Start PID_Compact' 按钮
节模式里, 选择默认 '预调节' 的 'start' 按钮, 如果后期发现不准,可以选择 '精确调节'
```

### 手动调试

```bash
大功率的加热,可以增大P
I很小或者为0的情况下, 如果比较接近目标, 则P可以以此为准；如果I为0, 超出目标,则需要减小P
积分时间越大,平缓度越大
微分时间越大,敏感度越高
```
