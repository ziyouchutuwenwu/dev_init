# 语法

## 基础用法

### PROGRAM

可以当作为模块，调用里面的 action 和 method
也可以直接作为函数被调用，默认变量为 static 变量，必须通过 VAR_INPUT 和 VAR_OUTPUT 作输入输出，调用的时候参数名必须传入

### FUNCTION_BLOCK

必须实例化，否则无法调用，默认变量为 static 变量, 必须通过 VAR_INPUT 和 VAR_OUTPUT 作输入输出，调用的时候参数名必须传入

### FUNCTION

普通函数调用，函数名作为返回值，也支持通过 VAR_INPUT 和 VAR_OUTPUT 作输入输出，调用的时候参数名必须传入.
建议变量使用 VAR_OUTPUT 的模式输出,这样可以使用不同的变量名,便于理解.如果有多个输出,再考虑函数返回值

## 注意

凡是涉及变量类型的，建议强制约束，避免错误

```bash
VAR_TEMP
VAR_STAT
```

## 面向对象

### 仅 FUNCTION_BLOCK 支持面向对象

同名 method，直接覆盖基 fb 的 method
关键字 this^, 可以访问 fb 的变量，fb 的属性
关键字 super^, 直接调用基 fb 的 method

#### interface 的多态流程

interface 定义 method
基 fb 实现 interface
子 fb 继承基 fb

##### 例子

```bash
demo :IDemo;
test_instance :BaseFB;
// test_instance :SonFB;
demo := test_instance;
demo.demoMethod();
```
