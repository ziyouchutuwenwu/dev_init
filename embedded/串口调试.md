# 串口调试说明

## 配置权限

- code 70-ttyUSB.rules

```bash
KERNEL=="ttyUSB[0-9]*",MODE="0666"
```

- sudo mv ./70-ttyUSB.rules /etc/udev/rules.d/

- 有些 micropython 板，接口是/dev/ttyACM 系列

```bash
KERNEL=="ttyACM[0-9]*",MODE="0666"
```

## 建议 picocom 的用法

```bash
#! /bin/bash

picocom -b 115200 -s "sb -vv" -v "rb -vv" /dev/ttyUSB0
```
