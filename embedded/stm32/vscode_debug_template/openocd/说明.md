# vscode 使用 openocd 调试 stm32

- openocd 需要加入 PATH 变量

```shell
export PATH=~/dev/embedded/stm32/openocd/bin/:$PATH
```

- 如果 vscode 提示错误，比如

```shell
Failed to launch GDB: Remote connection closed (from target-select extended-remote localhost:50000)
```

尝试手工命令行启动 openocd 加载配置文件，如果是 openocd Could not initialize the debug port 的错误，应该是 stm32CubeMX 生成的项目没有选择 sys，wire debug 模式
