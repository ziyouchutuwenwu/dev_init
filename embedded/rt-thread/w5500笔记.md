# W5500配置

## 物理接线

## cubeMX配置

- 启用spi1，同时设置 Hardware NSS Signal为output模式
- 生成项目，复制 Inc 和 Src 目录到项目，注释掉def.h里面的 exti 的宏定义

## menuconfig配置

- 配置 片上驱动，spi
- 配置iot packages 里面的 wiznet部分，注意，设备名不是 spi的总线名，是设备名，需要手工创建，根据下面的例子，设备名为spi10

## 编写spi初始化代码

```bash
#include <rtthread.h>
#include <rtdevice.h>
#include <board.h>

static int w5500_spi_init(void)
{
    __HAL_RCC_GPIOA_CLK_ENABLE();
    return rt_hw_spi_device_attach("spi1", "spi10", GPIOA, GPIO_PIN_4);
}
/* 导出到自动初始化 */
INIT_COMPONENT_EXPORT(w5500_spi_init);
```
