# W5500 配置

## 物理接线

## cubeMX 配置

- 启用 spi1，同时设置 Hardware NSS Signal 为 output 模式
- 生成项目，复制 Inc 和 Src 目录到项目，注释掉 def.h 里面的 exti 的宏定义

## menuconfig 配置

- 配置 片上驱动，spi
- 配置 iot packages 里面的 wiznet 部分，注意，设备名不是 spi 的总线名，是设备名，需要手工创建，根据下面的例子，设备名为 spi10

## 编写 spi 初始化代码

```bash
#include <rtthread.h>
#include <rtdevice.h>
#include <board.h>

static int w5500_spi_init(void)
{
    __HAL_RCC_GPIOA_CLK_ENABLE();
    return rt_hw_spi_device_attach("spi1", "spi10", GPIOA, GPIO_PIN_4);
}
/* 导出到自动初始化 */
INIT_COMPONENT_EXPORT(w5500_spi_init);
```
