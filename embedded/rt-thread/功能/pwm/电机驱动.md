# 步进电机的驱动

## 测试命令

```sh
假定周期为10*1000*1000=10000000
脉冲为10×0.5*1000*1000=5000000

pwm_enable pwm2 4
pwm_set pwm2 4 10000000 5000000
pwm_disable pwm2 4
```

## 计算公式

```txt
脉冲周期=1s/(每秒圈数*每圈脉冲数)
占空比一般50%
```

## 注意

同一个定时器，多个通道，目前 rtt 框架生成的 pwm 频率只能是一个，占空比可以不同

实际应用中,只需要在同一时刻会同时发的 pwm 使用不同的定时器即可.

脉冲经过光耦会失真
