# 虚拟串口

## 参考[这里](https://mp.weixin.qq.com/s/s8R4zg5-5HjxA4ldt0_n8g)

- 打开cubeMX，开启USB_OTG, 使用device模式，启用usb otg的全局中断，配置时钟，生成代码，复制到项目
  - 注意，cubeMX里面配置的时候，rcc需要启用hse，外部晶震，usb的时钟需要设置为48MHZ，我是F411CEU6的板
- 复制main.c里面的时钟配置到board.c
- 修改Kconfig，添加

```bash
    config BSP_USING_USBD_FS
        bool "Enable OTGFS as USB device"
        select RT_USING_USB_DEVICE
        select _RT_USB_DEVICE_CDC
        default n
```

- scons --menuconfig
  - Hardware Drivers Config > On-chip Peripheral Drivers， 启用usb otg
  - RT-Thread Components > Device Drivers > Using USB > Enable composite device, 选中 Enable to use device as CDC device

- 编译，烧录

- 测试代码

```c
#include <rtthread.h>
#include <rtdevice.h>

int vcom_test(void)
{
    rt_device_t dev = RT_NULL;
    char buf[] = "hello vcom!\r\n";

    dev = rt_device_find("vcom");

    if (dev)
        rt_device_open(dev, RT_DEVICE_FLAG_RDWR);
    else
        return -RT_ERROR;

    while (1)
    {
        rt_device_write(dev, 0, buf, rt_strlen(buf));
        rt_thread_mdelay(500);
    }

    rt_device_close(dev);
}

MSH_CMD_EXPORT(vcom_test, vcom_test)
```
