## i2c

- 参考配置

- 修改board/Kconfig，添加 i2c 的配置部分，以下为参考

```bash
    menuconfig BSP_USING_I2C1
        bool "Enable I2C1 BUS (software simulation)"
        default n
        select RT_USING_I2C
        select RT_USING_I2C_BITOPS
        select RT_USING_PIN
        if BSP_USING_I2C1
            config BSP_I2C1_SCL_PIN
                int "i2c1 scl pin number"
                range 1 216
                default 22
            config BSP_I2C1_SDA_PIN
                int "I2C1 sda pin number"
                range 1 216
                default 23
        endif
```

- scons --menuconfig 启用i2c
- stm32XXxx_hal_conf.h 需要启用 HAL_I2C_MODULE_ENABLED

## HWTIMER

- 修改board/Kconfig，添加 TIM 部分，以下为参考

```bash
menuconfig BSP_USING_TIM
        bool "enable timer"
        default n
        select RT_USING_HWTIMER
        if BSP_USING_TIM
            config BSP_USING_TIM11
                bool "Enable TIM11"
                default n
        endif
```

- scons --menuconfig 启用timer配置
- stm32XXxx_hal_conf.h 需要启用 HAL_TIM_MODULE_ENABLED

## spi

- 修改board/Kconfig，添加 SPI 部分，以下为参考

```bash
menuconfig BSP_USING_SPI
        bool "Enable SPI BUS"
        default n
        select RT_USING_SPI
        if BSP_USING_SPI
            config BSP_USING_SPI1
                bool "Enable SPI1 BUS"
                default n

            config BSP_SPI1_TX_USING_DMA
                bool "Enable SPI1 TX DMA"
                depends on BSP_USING_SPI1
                default n

            config BSP_SPI1_RX_USING_DMA
                bool "Enable SPI1 RX DMA"
                depends on BSP_USING_SPI1
                select BSP_SPI1_TX_USING_DMA
                default n
        endif
```

- scons --menuconfig 启用spi
- stm32XXxx_hal_conf.h 需要启用 HAL_SPI_MODULE_ENABLED
