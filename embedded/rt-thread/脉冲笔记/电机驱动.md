# 步进电机的驱动

- 脉冲正负都要接，并且一定要初始化，`就算使用pwm，也要初始化`

```C
rt_pin_mode(PUL_SUB_PIN, PIN_MODE_OUTPUT);
```

- 电机转动可以用gpio反复高低电平测试

```C
int main(void)
{
    int count = 1;

    rt_pin_mode(PUL_PLUS_PIN, PIN_MODE_OUTPUT);
    rt_pin_mode(PUL_SUB_PIN, PIN_MODE_OUTPUT);

    while (count++)
    {
        rt_pin_write(PUL_PLUS_PIN, PIN_HIGH);
        rt_thread_mdelay(5);
        rt_pin_write(PUL_PLUS_PIN, PIN_LOW);
        rt_thread_mdelay(5);
    }

    return RT_EOK;
}
```
