# 步进电机的驱动

- 脉冲正负都要接，并且一定要初始化，`就算使用pwm，也要初始化`

```C
rt_pin_mode(PUL_SUB_PIN, PIN_MODE_OUTPUT);
```

- 电机转动可以用gpio反复高低电平测试

```C
int main(void)
{
    int count = 1;

    rt_pin_mode(PUL_PLUS_PIN, PIN_MODE_OUTPUT);
    rt_pin_mode(PUL_SUB_PIN, PIN_MODE_OUTPUT);

    while (count++)
    {
        rt_pin_write(PUL_PLUS_PIN, PIN_HIGH);
        rt_thread_mdelay(5);
        rt_pin_write(PUL_PLUS_PIN, PIN_LOW);
        rt_thread_mdelay(5);
    }

    return RT_EOK;
}
```

- 生产使用 pwm

- 测试命令

```bash
pwm_enable pwm2 4
pwm_set pwm2 4 10*1000*1000 5*1000*1000
pwm_disable pwm2 4
```

- 算法

```c
细分数为1, 一圈200个脉冲
假定需要2s，每秒脉冲100个
脉冲周期为10ms，占空比一般50%
则测试代码为
```

```C
#include <rtthread.h>
#include <rtdevice.h>

#define PWM_DEV_NAME        "pwm2"
#define PWM_DEV_CHANNEL     4

struct rt_device_pwm *pwm_dev;

static int pwm_demo(int argc, char *argv[])
{
    rt_uint32_t period, pulse, dir;

    period = 10 * 1000 * 1000;
    pulse = 5 * 1000 * 1000;

    pwm_dev = (struct rt_device_pwm *)rt_device_find(PWM_DEV_NAME);
    if (pwm_dev == RT_NULL)
    {
        rt_kprintf("pwm demo run failed! can't find %s device!\n", PWM_DEV_NAME);
        return RT_ERROR;
    }

    rt_pwm_set(pwm_dev, PWM_DEV_CHANNEL, period, pulse);
    rt_pwm_enable(pwm_dev, PWM_DEV_CHANNEL);

    rt_thread_mdelay(2*1000);
    rt_pwm_disable(pwm_dev, PWM_DEV_CHANNEL);
}

MSH_CMD_EXPORT(pwm_demo, pwm demo);
```

- 如果发现频率不对，可能需要自己调整脉冲周期或者计数器的个数
- 如果需要精确的计数，建议使用硬件定时器来控制电机，演示代码见 step_motor
