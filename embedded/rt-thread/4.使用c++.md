# 以下为流程

- 参考[Wiki]([url](https://www.rt-thread.org/document/site/application-note/components/cplusplus/an0035-cpp/))，略有不同

```bash
cd rt-thread/bsp/qemu-vexpress-a9/
tree
├── applications
│   ├── lcd_init.c
│   ├── main.cpp
│   ├── mnt.c
│   ├── my_lib
│   │   ├── my_lib.cpp
│   │   └── my_lib.h
│   └── SConscript
```

- my_lib.h

```cpp
#include <iostream>

class MyLib{
public:
    void demo();
};
```

- my_lib.cpp

```cpp
#include "my_lib.h"

void MyLib::demo(){
    printf("hello demo in my lib\n");
}
```

- main.cpp

```cpp
#include "my_lib/my_lib.h"

int main(void){
    MyLib* lib = new MyLib;
    lib->demo();

    return 0;
}
```

- SConscript添加

```bash
src += ['my_lib/my_lib.cpp']
group = DefineGroup('my_lib', src, depend = [''], CPPPATH = CPPPATH)
```
