# 过程

- 下载源码，安装编译器

```bash
git clone https://github.com/RT-Thread/rt-thread.git
sudo apt install scons
```

- 下载gcc编译器，如果需要支持pthread，需要使用老的版本，这个版本是支持pthread的最后一个版本

```bash
https://armkeil.blob.core.windows.net/developer/Files/downloads/gnu-rm/6-2016q4/gcc-arm-none-eabi-6_2-2016q4-20161216-linux.tar.bz2
https://armkeil.blob.core.windows.net/developer/Files/downloads/gnu-rm/6-2017q2/gcc-arm-none-eabi-6-2017-q2-update-linux.tar.bz2
```

- 修改编译配置

```bash
rt-thread/bsp/qemu-vexpress-a9/rtconfig.py
修改gcc的路径
EXEC_PATH   = '/home/mmc/dev/embedded/rtt/gcc-arm-none-eabi-6_2-2016q4/bin/'
```

- 编译

```bash
cd rt-thread/bsp/qemu-vexpress-a9/
scons -c; scons --menuconfig; scons
```
